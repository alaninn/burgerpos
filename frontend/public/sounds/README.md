# Sonidos de Notificación

Para una mejor experiencia, reemplaza estos archivos con sonidos premium de alta calidad.

## Fuentes recomendadas de sonidos gratuitos:

1. **Freesound.org** (licencia CC0)
   - https://freesound.org/search/?q=notification

2. **Zapsplat.com** (cuenta gratuita)
   - https://www.zapsplat.com/sound-effect-categories/notification-sounds/

3. **Notification Sounds**
   - https://notificationsounds.com/

## Archivos necesarios:

- `alert1.mp3` - Notificación suave (ding elegante)
- `alert2.mp3` - Campana cristalina
- `alert3.mp3` - Ding corto y nítido
- `alert4.mp3` - Pop moderno
- `alert5.mp3` - Notificación de smartphone
- `alert6.mp3` - Timbre de servicio

## Especificaciones técnicas:

- Formato: MP3
- Duración: 0.5 - 2 segundos
- Bitrate: 128kbps o superior
- Volumen normalizado
